# 센서 교정 가이드

## 1. 온도 센서 (Temperature Sensors)

### Temperature1 - 용탕 온도 센서
- **위치**: 용해로 내부
- **타입**: K-type 열전대
- **측정 범위**: 0-800°C
- **정확도**: ±2°C
- **교정 주기**: 월 1회

#### 교정 방법
1. 표준 온도계와 비교 측정
2. 3점 교정 (200°C, 400°C, 650°C)
3. 편차 ±5°C 초과 시 센서 교체

### Temperature2 - 금형 온도 센서
- **위치**: 금형 고정측
- **타입**: K-type 열전대
- **측정 범위**: 0-400°C
- **정확도**: ±2°C
- **교정 주기**: 월 1회

### Temperature3 - 냉각수 온도 센서
- **위치**: 냉각수 출구
- **타입**: PT100 RTD
- **측정 범위**: 0-100°C
- **정확도**: ±0.5°C
- **교정 주기**: 분기 1회

## 2. 압력 센서 (Pressure Sensors)

### Pressure1 - 사출 압력 센서
- **위치**: 사출 실린더
- **측정 범위**: 0-200 MPa
- **정확도**: ±1%
- **교정 주기**: 월 1회

#### 권장 범위
- 1차 사출: 20-40 MPa
- 2차 사출: 80-120 MPa
- 증압: 120-150 MPa

### Pressure2 - 형체 압력 센서
- **위치**: 형체 실린더
- **측정 범위**: 0-30 MPa
- **정확도**: ±1%
- **교정 주기**: 분기 1회

### Pressure3 - 유압 시스템 압력
- **위치**: 유압 펌프 출구
- **측정 범위**: 0-20 MPa
- **정확도**: ±0.5%
- **교정 주기**: 분기 1회

## 3. 속도/위치 센서

### Velocity - 사출 속도 센서
- **타입**: 선형 인코더
- **측정 범위**: 0-10 m/s
- **분해능**: 0.01 m/s
- **교정 주기**: 분기 1회

#### 권장 사출 속도
- 저속 사출: 0.1-0.5 m/s
- 고속 사출: 2-6 m/s

### Position - 플런저 위치 센서
- **타입**: 마그네틱 스케일
- **측정 범위**: 0-700 mm
- **분해능**: 0.1 mm
- **교정 주기**: 반기 1회

## 4. 진동 센서 (Vibration Sensor)

- **위치**: 다이캐스팅 머신 베이스
- **타입**: 가속도계
- **측정 범위**: 0-50 mm/s (RMS)
- **주파수 범위**: 10-1000 Hz
- **교정 주기**: 반기 1회

### 진동 기준값
| 상태 | 진동값 (mm/s) | 조치 |
|------|---------------|------|
| 양호 | 0-2.8 | 정상 운전 |
| 주의 | 2.8-7.1 | 모니터링 강화 |
| 경고 | 7.1-18 | 점검 필요 |
| 위험 | >18 | 즉시 정지 |

## 6. 센서 이상 시 조치

### 온도 센서 이상
1. 배선 연결 상태 확인
2. 열전대 접점 상태 확인
3. 보상 도선 확인
4. 센서 교체

### 압력 센서 이상
1. 배선 확인
2. 영점 조정
3. 스팬 조정
4. 센서 교체
