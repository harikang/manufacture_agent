# 다이캐스팅 공정 표준 작업 절차 (SOP)

## 1. 공정 개요
다이캐스팅은 용융된 금속을 고압으로 금형에 주입하여 제품을 성형하는 공정입니다.

## 2. 작업 전 준비사항

### 2.1 장비 점검
- 금형 온도: 150-200°C 범위 확인
- 유압 시스템 압력: 정상 범위 확인
- 냉각수 순환: 정상 작동 확인
- 안전 장치: 작동 상태 확인

### 2.2 재료 준비
- 용탕 온도: 650-680°C 유지
- 용탕 품질: 불순물 제거 확인
- 이형제: 적정량 준비

## 3. 작업 절차

### 3.1 금형 준비
1. 금형 청소 및 이형제 도포
2. 금형 온도 확인 (180°C ± 10°C)
3. 금형 체결 및 클램프 압력 확인 (800 kN)

### 3.2 사출 작업
1. 용탕 온도 확인 (650-680°C)
2. 사출 속도 설정 (2.0-3.0 m/s)
3. 사출 압력 설정 (120-130 MPa)
4. 사출 시간: 1.0-1.5초
5. 보압 시간: 2.5-3.5초
6. 보압 압력: 85-95 MPa

### 3.3 냉각 및 취출
1. 냉각 시간: 15-20초
2. 금형 개방
3. 제품 취출
4. 제품 검사

## 4. 품질 관리 포인트

### 4.1 온도 관리
- 용탕 온도: ±5°C 이내 유지
- 금형 온도: ±10°C 이내 유지
- 온도 편차 발생 시 즉시 조정

### 4.2 압력 관리
- 사출 압력: 설정값 ±5% 이내
- 보압 압력: 설정값 ±5% 이내
- 압력 이상 시 즉시 작업 중단

### 4.3 시간 관리
- 사이클 타임: 40-50초 유지
- 냉각 시간 부족 시 변형 발생 가능

## 5. 이상 발생 시 조치

### 5.1 온도 이상
- 용탕 온도 과다: 냉각 대기 후 재작업
- 금형 온도 과다: 냉각수 유량 증가

### 5.2 압력 이상
- 압력 부족: 유압 시스템 점검
- 압력 과다: 설정값 재확인

### 5.3 불량 발생
- 기공 발생: 사출 속도/압력 조정
- 변형 발생: 냉각 시간 증가
- 표면 불량: 금형 청소 및 이형제 재도포

## 6. 안전 수칙
- 고온 용탕 취급 시 보호구 착용 필수
- 금형 작업 시 안전 거리 유지
- 비상 정지 버튼 위치 숙지
- 작업 종료 후 장비 전원 차단

## 7. 기록 관리
- 작업 일지 작성
- 불량률 기록
- 장비 점검 기록
- 이상 발생 시 보고서 작성
