## 1. 목적 및 적용 범위

본 지침서는 다이캐스팅 주입 공정의 표준 작업 절차를 정의하여 제품 품질의 일관성을 확보하고 안전사고를 예방하는 것을 목적으로 합니다.

**적용 범위**: 모든 알루미늄 다이캐스팅 주입 공정

## 2. 안전 수칙

### 2.1 필수 보호구
- 내열 장갑 (최소 300°C 내열)
- 안전화 (철심 포함)
- 보안경
- 내열 앞치마
- 안전모

### 2.2 금지 사항
- ⚠️ 용탕 근처에서 물 사용 금지
- ⚠️ 젖은 도구 사용 금지
- ⚠️ 금형 작동 중 손 접근 금지
- ⚠️ 안전 가드 제거 금지

## 3. 작업 전 준비

### 3.1 장비 점검 (체크리스트)

#### 주조기 점검
- [ ] 유압 시스템 압력 확인 (150-180 bar)
- [ ] 윤활유 레벨 확인
- [ ] 비상정지 버튼 작동 확인
- [ ] 안전 가드 상태 확인
- [ ] 냉각수 순환 확인

#### 금형 점검
- [ ] 금형 표면 청결 상태 확인
- [ ] 배기구 막힘 여부 확인
- [ ] 이젝터 핀 작동 확인
- [ ] 온도 센서 연결 확인
- [ ] 냉각 채널 누수 확인

#### 용해로 점검
- [ ] 용탕 온도 확인 (660-680°C)
- [ ] 용탕 레벨 확인
- [ ] 슬래그 제거
- [ ] 탈가스 처리 완료 확인

### 3.2 금형 예열

**목표 온도**: 180-200°C

```
예열 절차:
1. 냉각수 온도를 80°C로 설정
2. 금형 히터 가동 (30분)
3. 온도 센서로 각 부위 온도 확인
   - 캐비티 부: 190-200°C
   - 코어 부: 180-190°C
   - 슬라이드 부: 170-180°C
4. 온도 편차 ±10°C 이내 확인
```

### 3.3 이형제 준비

**이형제 타입**: 수용성 이형제 (희석 비율 1:20)

```
이형제 준비:
1. 이형제 농도 확인 (굴절계 사용)
2. 분무 압력 설정 (3-4 bar)
3. 분무 패턴 테스트
4. 건조 시간 확인 (15-20초)
```

## 4. 주입 공정 표준 파라미터

### 4.1 기본 파라미터

| 파라미터 | 설정값 | 허용 범위 | 모니터링 센서 |
|---------|--------|----------|--------------|
| 용탕 온도 | 670°C | 660-680°C | Sensor_Temperature1 |
| 금형 온도 | 190°C | 180-200°C | Sensor_Temperature2 |
| 주입 속도 | 2.5 m/s | 2.0-2.8 m/s | Sensor_Speed |
| 주입 압력 | 120 bar | 110-130 bar | Sensor_Pressure1 |
| 보압 압력 | 90 bar | 80-100 bar | Sensor_Pressure2 |
| 보압 시간 | 3.0 sec | 2.5-3.5 sec | Process_HoldTime |
| 냉각 시간 | 15 sec | 12-18 sec | Process_CoolingTime |
| 사이클 타임 | 45 sec | 40-50 sec | Process_CycleTime |

### 4.2 파라미터 조정 가이드

#### 제품 두께별 조정
- **얇은 제품 (< 2mm)**:
  - 주입 속도: +10%
  - 금형 온도: +10°C
  - 냉각 시간: -20%

- **두꺼운 제품 (> 5mm)**:
  - 주입 속도: -10%
  - 보압 시간: +30%
  - 냉각 시간: +30%

#### 재질별 조정
- **ADC12 (일반)**: 표준 파라미터 사용
- **ADC10 (고강도)**: 용탕 온도 +10°C
- **ADC1 (고순도)**: 주입 속도 -10%

## 5. 작업 절차

### 5.1 첫 샷 (First Shot)

```
Step 1: 금형 이형제 도포
- 분무 거리: 30-40cm
- 분무 시간: 3-5초
- 건조 시간: 15-20초
- 확인: 균일한 피막 형성

Step 2: 금형 클로징
- 클램핑 압력: 800 ton
- 클로징 속도: 저속 → 고속 전환
- 안전 거리 확인: 50mm

Step 3: 용탕 주입
- 래들 온도 확인
- 슬래그 제거 확인
- 주입량 확인: 250g ± 10g
- 주입 시작

Step 4: 냉각 및 개방
- 냉각 시간 준수
- 금형 개방
- 제품 이젝트

Step 5: 첫 샷 검사
- 외관 검사
- 치수 검사 (주요 치수 3개)
- 불량 여부 판정
- 합격 시 정상 생산 시작
```

### 5.2 정상 생산

```
반복 사이클:
1. 이형제 도포 (매 샷)
2. 금형 클로징
3. 용탕 주입
4. 냉각 (15초)
5. 금형 개방
6. 제품 이젝트
7. 제품 취출
8. 육안 검사
9. 다음 사이클 시작

주기적 점검:
- 매 10 shot: 치수 측정
- 매 50 shot: 금형 온도 확인
- 매 100 shot: 배기구 점검
- 매 shift: 전수 검사 샘플링
```

### 5.3 이상 발생 시 대응

#### 불량 발생 시
```
1. 즉시 생산 중단
2. 불량 유형 확인
   - 포로시티: 주입 속도/배압 조정
   - 크랙: 금형 온도 상승
   - 치수 불량: 금형 점검
3. 파라미터 조정
4. 테스트 샷 3회
5. 품질 확인 후 재개
```

#### 장비 이상 시
```
1. 비상정지 버튼 작동
2. 안전 확인
3. 유지보수팀 호출
4. 이상 내용 기록
5. 수리 완료 후 재가동 절차 진행
```

## 6. 품질 관리

### 6.1 공정 중 검사

**검사 항목**:
- 외관 검사 (전수)
- 치수 검사 (매 10 shot)
- 중량 검사 (매 50 shot)
- X-ray 검사 (매 100 shot)

**합격 기준**:
- 외관: 크랙, 기공, 변형 없음
- 치수: 도면 공차 이내
- 중량: ±5% 이내

### 6.2 데이터 기록

**기록 항목**:
- 생산 수량 (양품/불량)
- 공정 파라미터 (시간별)
- 불량 유형 및 원인
- 조치 사항
- 작업자 정보

**기록 주기**: 매 시간

## 7. 작업 종료

### 7.1 종료 절차

```
Step 1: 생산 중단
- 마지막 제품 취출
- 금형 개방 상태 유지

Step 2: 금형 청소
- 이형제 잔류물 제거
- 배기구 청소
- 이젝터 핀 청소

Step 3: 금형 방청
- 방청유 도포
- 금형 클로징

Step 4: 장비 정리
- 용탕 처리
- 냉각수 배수 (동절기)
- 유압 시스템 OFF
- 전원 차단

Step 5: 작업 일지 작성
- 생산 실적 기록
- 불량 현황 기록
- 특이사항 기록
- 다음 작업자 인계사항 기록
```

## 8. 비상 상황 대응

### 8.1 화재 발생 시
1. 비상벨 작동
2. 소화기 사용 (ABC 분말 소화기)
3. 대피 (비상구 이용)
4. 119 신고

### 8.2 용탕 누출 시
1. 즉시 대피 (반경 5m)
2. 비상정지 버튼 작동
3. 안전 담당자 호출
4. 냉각 후 처리

### 8.3 부상 발생 시
1. 응급처치
2. 의무실 연락
3. 필요시 119 신고
4. 사고 보고서 작성
