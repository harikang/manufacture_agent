# 다이캐스팅 작업장 안전 규정

**문서 메타데이터:**
- Category: regulations
- Regulation Type: safety
- Authority: KOSHA (한국산업안전보건공단)
- Effective Date: 2024-01-01
- Compliance Level: mandatory
- Version: 2.0

## 1. 법적 근거

본 규정은 다음 법령에 근거하여 제정되었습니다:
- 산업안전보건법 제38조 (안전조치)
- 산업안전보건법 제39조 (보건조치)
- 산업안전보건기준에 관한 규칙 제3편 제2장 (주조작업)

## 2. 적용 범위

본 규정은 다이캐스팅 작업장 내 모든 작업자, 관리자, 방문자에게 적용됩니다.

## 3. 개인 보호구 (PPE) 착용 의무

### 3.1 필수 보호구

#### 작업자
1. **안전모** (KCS 인증)
   - 착용 시기: 작업장 출입 시 항상
   - 교체 주기: 2년 또는 충격 발생 시 즉시
   - 관련 법규: 산업안전보건기준에 관한 규칙 제32조

2. **안전화** (KCS 인증, 철심 포함)
   - 착용 시기: 작업장 출입 시 항상
   - 내열 온도: 최소 200°C
   - 교체 주기: 6개월 또는 손상 시 즉시

3. **보안경** (내열성, 비산물 방지)
   - 착용 시기: 용탕 취급 작업 시 필수
   - 규격: KS M 6805
   - 교체 주기: 1년 또는 스크래치 발생 시

4. **내열 장갑** (300°C 이상 내열)
   - 착용 시기: 용탕 취급, 금형 작업 시
   - 재질: 아라미드 섬유 또는 가죽
   - 교체 주기: 3개월 또는 손상 시 즉시

5. **내열 앞치마**
   - 착용 시기: 용탕 주입 작업 시
   - 길이: 무릎 아래까지
   - 재질: 알루미늄 코팅 섬유

### 3.2 선택적 보호구

- **방진 마스크**: 연마 작업 시
- **귀마개**: 소음 85dB 이상 구역
- **용접 보안면**: 용접 작업 시

### 3.3 보호구 미착용 시 처벌

| 위반 횟수 | 조치 사항 |
|----------|----------|
| 1차 | 구두 경고 |
| 2차 | 서면 경고 |
| 3차 | 작업 중지 (1일) |
| 4차 | 징계 위원회 회부 |

## 4. 작업장 안전 수칙

### 4.1 용탕 취급 안전

#### 금지 사항 (위반 시 즉시 작업 중지)

1. **물과의 접촉 금지**
   - ⚠️ 용탕 근처에서 물 사용 절대 금지
   - ⚠️ 젖은 도구 사용 금지
   - ⚠️ 젖은 장갑 착용 금지
   - **위험**: 수증기 폭발로 인한 화상, 실명

2. **안전 거리 유지**
   - 용탕 취급 시 최소 1m 거리 유지
   - 주입 중 금형 전면 접근 금지
   - 비작업자 출입 금지 구역 설정 (반경 3m)

3. **적절한 복장**
   - 합성섬유 의류 착용 금지 (용융 위험)
   - 면 또는 난연 소재 작업복 착용
   - 반바지, 샌들 착용 금지

#### 용탕 온도 관리

```
안전 온도 범위: 660-700°C
- 최저 온도: 660°C (미만 시 주입 불가)
- 최고 온도: 700°C (초과 시 냉각 후 사용)
- 측정 주기: 매 30분
- 기록 의무: 매 시간
```

#### 용탕 이송 안전

1. 이송 전 확인사항
   - [ ] 이송 경로 장애물 제거
   - [ ] 바닥 물기 제거
   - [ ] 래들 상태 확인 (균열, 손상)
   - [ ] 주변 작업자 대피 확인

2. 이송 중 준수사항
   - 천천히 이동 (최대 속도: 0.5 m/s)
   - 용탕 흔들림 최소화
   - 이송 경로 외 우회 금지
   - 이송 중 대화 금지

### 4.2 장비 안전

#### 주조기 안전

1. **안전 장치 필수 사항**
   - 비상정지 버튼: 작동 확인 (매일)
   - 안전 가드: 제거 금지
   - 양손 조작 장치: 정상 작동 확인
   - 광전자 센서: 정상 작동 확인

2. **작동 중 금지 사항**
   - ⚠️ 금형 내부 손 접근 금지
   - ⚠️ 안전 가드 제거 금지
   - ⚠️ 작동 중 청소 금지
   - ⚠️ 비정상 소음 발생 시 즉시 중단

3. **정기 점검**
   - 일일 점검: 안전 장치 작동 확인
   - 주간 점검: 유압 시스템, 전기 시스템
   - 월간 점검: 전문가 정밀 점검
   - 연간 점검: 법정 안전 검사

#### 금형 안전

1. **금형 교체 작업**
   - 작업 전 전원 차단 및 잠금 (LOTO)
   - 크레인 사용 시 신호수 배치
   - 금형 하부 출입 금지
   - 안전 블록 사용

2. **금형 가열 중 안전**
   - 고온 표시 부착
   - 접근 금지 구역 설정
   - 화상 위험 경고

### 4.3 작업장 환경 안전

#### 환기 기준

```
법적 기준 (산업안전보건기준에 관한 규칙 제618조):
- 환기 횟수: 시간당 10회 이상
- CO 농도: 30 ppm 이하
- 분진 농도: 10 mg/m³ 이하
- 측정 주기: 월 1회
```

#### 소음 관리

```
소음 기준 (산업안전보건기준에 관한 규칙 제512조):
- 85 dB 이상: 귀마개 착용 의무
- 90 dB 이상: 소음 저감 조치 필수
- 측정 주기: 분기 1회
- 작업 시간 제한: 90 dB 이상 시 8시간 이하
```

#### 조도 기준

```
작업별 조도 기준 (KS A 3011):
- 정밀 작업: 750 lux 이상
- 일반 작업: 300 lux 이상
- 통로: 150 lux 이상
- 측정 주기: 분기 1회
```

## 5. 비상 상황 대응

### 5.1 화재 발생 시

#### 초기 대응 (발견 후 1분 이내)

```
Step 1: 비상벨 작동
- 위치: 작업장 4개소
- 작동 방법: 버튼 누름

Step 2: 초기 진화 시도
- 소화기 종류: ABC 분말 소화기
- 사용 방법: PASS (Pull-Aim-Squeeze-Sweep)
- 진화 불가 시 즉시 대피

Step 3: 대피
- 대피 경로: 비상구 (2개소)
- 집결지: 주차장
- 인원 확인

Step 4: 신고
- 119 신고
- 안전 관리자 보고
```

#### 금속 화재 (마그네슘, 알루미늄)

- ⚠️ 물 사용 절대 금지
- D급 소화기 사용 (금속 화재용)
- 모래 또는 건조 분말 사용
- 전문 소방대 대기

### 5.2 용탕 누출 사고

```
Step 1: 즉시 대피 (반경 5m)
Step 2: 비상정지 버튼 작동
Step 3: 안전 관리자 호출
Step 4: 냉각 대기 (최소 30분)
Step 5: 안전 확인 후 처리
```

### 5.3 부상 발생 시

#### 화상

```
경미한 화상 (1도):
1. 즉시 흐르는 물로 냉각 (10-15분)
2. 화상 연고 도포
3. 의무실 방문

중증 화상 (2도 이상):
1. 즉시 흐르는 물로 냉각
2. 깨끗한 천으로 덮기
3. 119 신고
4. 의무실 연락
5. 사고 보고서 작성
```

#### 골절, 타박상

```
1. 환자 안정
2. 부상 부위 고정
3. 의무실 연락
4. 필요시 119 신고
5. 사고 보고서 작성
```

## 6. 안전 교육

### 6.1 교육 의무

| 교육 종류 | 대상 | 주기 | 시간 |
|----------|------|------|------|
| 신규 채용 교육 | 신입 사원 | 입사 시 | 8시간 |
| 정기 안전 교육 | 전 직원 | 분기 1회 | 2시간 |
| 특별 안전 교육 | 관리 감독자 | 연 1회 | 16시간 |
| 작업 내용 변경 교육 | 해당 작업자 | 변경 시 | 2시간 |

### 6.2 교육 내용

1. **법령 및 규정**
   - 산업안전보건법
   - 사내 안전 규정
   - 작업 표준

2. **위험성 평가**
   - 작업별 위험 요인
   - 사고 사례
   - 예방 대책

3. **비상 대응**
   - 화재 대응
   - 응급처치
   - 대피 훈련

## 7. 위험성 평가

### 7.1 평가 주기

- 정기 평가: 연 1회
- 수시 평가: 작업 변경 시, 사고 발생 시
- 평가 방법: 4M (Man, Machine, Material, Method)

### 7.2 위험 등급

| 등급 | 위험도 | 조치 사항 |
|------|--------|----------|
| 상 | 15-25 | 즉시 개선 (작업 중지) |
| 중 | 10-14 | 1개월 이내 개선 |
| 하 | 5-9 | 3개월 이내 개선 |

## 8. 사고 보고 및 조사

### 8.1 보고 의무

```
중대 재해 (사망, 3일 이상 입원):
- 즉시 보고 (1시간 이내)
- 고용노동부 신고
- 작업 중지

일반 재해 (3일 미만 치료):
- 당일 보고
- 사고 조사
- 재발 방지 대책
```

### 8.2 조사 절차

1. 사고 현장 보존
2. 목격자 진술 확보
3. 원인 분석 (5 Why 기법)
4. 재발 방지 대책 수립
5. 수평 전개 (유사 작업장)
