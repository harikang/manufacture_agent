# 다이캐스팅 품질 기준서

## 1. 품질 등급 분류

### 양품 (Normal)
- 모든 검사 항목 합격
- 외관 결함 없음
- 치수 공차 이내
- 기계적 특성 만족

### 불량 (Defect)
- 1개 이상의 검사 항목 불합격
- 외관 결함 존재
- 치수 공차 초과
- 기계적 특성 미달

## 2. 주요 불량 유형

### 2.1 기공 (Porosity)
**정의**: 제품 내부 또는 표면에 발생하는 공극

**원인**:
- 용탕 온도 과다 (>700°C)
- 사출 속도 과다 (>6 m/s)
- 금형 배기 불량
- 이형제 과다 도포

**판정 기준**:
| 등급 | 기공 크기 | 기공 수 | 판정 |
|------|-----------|---------|------|
| A | <0.5mm | <3개/cm² | 합격 |
| B | 0.5-1mm | <2개/cm² | 조건부 합격 |
| C | >1mm | >2개/cm² | 불합격 |

### 2.2 수축 (Shrinkage)
**정의**: 응고 수축으로 인한 함몰 또는 내부 공동

**원인**:
- 증압 부족 (<100 MPa)
- 증압 시간 부족
- 금형 온도 불균일
- 게이트 조기 응고

**판정 기준**:
- 표면 수축 깊이 <0.3mm: 합격
- 내부 수축 직경 <2mm: 합격

### 2.3 미성형 (Short Shot)
**정의**: 용탕이 금형 캐비티를 완전히 채우지 못함

**원인**:
- 용탕 온도 부족 (<650°C)
- 사출 속도 부족
- 금형 온도 부족
- 용탕량 부족

**판정 기준**: 미성형 발생 시 전량 불합격

### 2.4 균열 (Crack)
**정의**: 제품 표면 또는 내부의 갈라짐

**원인**:
- 금형 온도 과다
- 취출 시기 부적절
- 이젝터 핀 위치 불량
- 제품 설계 문제

**판정 기준**: 균열 발생 시 전량 불합격

### 2.5 플래시 (Flash)
**정의**: 금형 분리면에서 용탕이 새어나온 것

**원인**:
- 형체력 부족
- 금형 마모
- 사출 압력 과다
- 금형 정렬 불량

**판정 기준**:
- 플래시 두께 <0.3mm: 후가공 후 합격
- 플래시 두께 >0.3mm: 불합격

## 3. 치수 검사 기준

### 일반 공차 (ISO 8062)
| 기본 치수 | CT6 | CT7 | CT8 |
|-----------|-----|-----|-----|
| ~25mm | ±0.38 | ±0.52 | ±0.74 |
| 25-40mm | ±0.42 | ±0.58 | ±0.82 |
| 40-63mm | ±0.46 | ±0.64 | ±0.90 |
| 63-100mm | ±0.52 | ±0.72 | ±1.00 |

### 중요 치수
- 조립부: CT6 적용
- 기능부: CT7 적용
- 일반부: CT8 적용

## 4. 외관 검사 기준

### 검사 조건
- 조명: 500 lux 이상
- 거리: 30cm
- 시간: 10초/면

### 판정 기준
| 결함 유형 | 허용 기준 |
|-----------|-----------|
| 스크래치 | 길이 5mm 이하, 깊이 0.1mm 이하 |
| 찍힘 | 직경 1mm 이하, 깊이 0.2mm 이하 |
| 변색 | 불허 |
| 이물질 | 불허 |

## 5. 기계적 특성 기준

### 알루미늄 합금 ADC12
| 항목 | 기준값 | 단위 |
|------|--------|------|
| 인장강도 | ≥280 | MPa |
| 항복강도 | ≥140 | MPa |
| 연신율 | ≥1.5 | % |
| 경도 | 75-95 | HB |

## 6. 샘플링 검사 기준

### 검사 수준
- 초기 양산: 전수 검사
- 안정 양산: AQL 1.0 (MIL-STD-1916)

### 검사 빈도
| 검사 항목 | 빈도 |
|-----------|------|
| 외관 검사 | 매 로트 |
| 치수 검사 | 매 로트 (n=5) |
| X-ray 검사 | 주 1회 |
| 기계적 시험 | 월 1회 |
