## 2. 파라미터 영향도 분석

### 품질에 미치는 영향 (중요도 순)

1. **용탕 온도 (Temperature1)** - 영향도: 25%
   - 높을 때: 기공 증가, 금형 수명 감소
   - 낮을 때: 미성형, 콜드샷 발생

2. **사출 속도 (Velocity)** - 영향도: 20%
   - 높을 때: 기공 증가, 플래시 발생
   - 낮을 때: 미성형, 표면 불량

3. **증압 (Pressure2)** - 영향도: 18%
   - 높을 때: 플래시, 금형 손상
   - 낮을 때: 수축, 기공 증가

4. **금형 온도 (Temperature2)** - 영향도: 15%
   - 높을 때: 사이클 증가, 수축
   - 낮을 때: 미성형, 표면 불량

5. **냉각 시간** - 영향도: 12%
   - 길 때: 생산성 저하
   - 짧을 때: 변형, 균열

## 3. 불량 유형별 파라미터 조정

### 기공 (Porosity) 발생 시
```
조치 우선순위:
1. 용탕 온도 ↓ (670°C → 660°C)
2. 사출 속도 ↓ (4.0 → 3.5 m/s)
3. 진공 배기 확인
4. 이형제 농도 ↓
```

### 수축 (Shrinkage) 발생 시
```
조치 우선순위:
1. 증압 ↑ (120 → 130 MPa)
2. 증압 시간 ↑ (5 → 7 sec)
3. 금형 온도 균일화
4. 게이트 크기 확인
```

### 미성형 (Short Shot) 발생 시
```
조치 우선순위:
1. 용탕 온도 ↑ (670°C → 680°C)
2. 사출 속도 ↑ (4.0 → 4.5 m/s)
3. 금형 온도 ↑ (180°C → 200°C)
4. 용탕량 확인
```

### 플래시 (Flash) 발생 시
```
조치 우선순위:
1. 형체력 확인
2. 사출 압력 ↓
3. 금형 분리면 점검
4. 금형 정렬 확인
```

## 4. 공정 능력 지수 (Cpk) 목표

| 파라미터 | 목표 Cpk | 현재 관리 수준 |
|----------|----------|----------------|
| 용탕 온도 | ≥1.33 | ±10°C |
| 금형 온도 | ≥1.33 | ±10°C |
| 사출 속도 | ≥1.50 | ±0.3 m/s |
| 증압 | ≥1.33 | ±5 MPa |
| 사이클 타임 | ≥1.67 | ±3 sec |

## 5. 이상 발생 시 대응 절차

### Level 1: 경미한 이상
- Cpk 1.0-1.33
- 조치: 파라미터 미세 조정
- 보고: 작업일지 기록

### Level 2: 중간 이상
- Cpk 0.67-1.0
- 조치: 공정 중단 후 원인 분석
- 보고: 반장에게 보고

### Level 3: 심각한 이상
- Cpk <0.67
- 조치: 즉시 생산 중단
- 보고: 품질팀 및 생산팀장 보고
