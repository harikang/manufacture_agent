# 다이캐스팅 불량 원인 분석 및 해결 방안

## 1. 기공 (Porosity)

### 1.1 원인
- **가스 기공**
  - 용탕 내 가스 함유량 과다
  - 사출 속도 과다로 인한 난류 발생
  - 금형 배기 불량
  
- **수축 기공**
  - 냉각 속도 불균형
  - 보압 부족
  - 용탕 온도 과다

### 1.2 해결 방안
1. **사출 조건 조정**
   - 사출 속도: 2.5 m/s → 2.2 m/s로 감소
   - 사출 압력: 120 MPa → 125 MPa로 증가
   - 보압 시간: 3.0초 → 3.5초로 증가

2. **금형 개선**
   - 배기구 청소 및 확대
   - 오버플로우 웰 추가

3. **용탕 관리**
   - 탈가스 처리 강화
   - 용탕 온도: 680°C → 670°C로 감소

## 2. 변형 (Warpage)

### 2.1 원인
- 냉각 불균형
- 금형 온도 편차
- 취출 시기 부적절
- 잔류 응력

### 2.2 해결 방안
1. **냉각 최적화**
   - 냉각 시간: 15초 → 18초로 증가
   - 냉각수 유량 균등 분배
   - 금형 온도 편차 ±5°C 이내 유지

2. **취출 조건**
   - 충분한 냉각 후 취출
   - 취출 속도 조정
   - 이젝터 핀 위치 최적화

## 3. 표면 불량 (Surface Defects)

### 3.1 콜드 샷 (Cold Shot)
**원인**
- 용탕 온도 부족
- 사출 속도 부족
- 금형 온도 부족

**해결 방안**
- 용탕 온도: 650°C → 665°C로 증가
- 사출 속도: 2.3 m/s → 2.6 m/s로 증가
- 금형 예열 강화

### 3.2 플로우 마크 (Flow Mark)
**원인**
- 사출 속도 불균일
- 금형 온도 낮음
- 이형제 과다

**해결 방안**
- 사출 속도 프로파일 최적화
- 금형 온도: 175°C → 185°C로 증가
- 이형제 도포량 감소

### 3.3 표면 거칠기
**원인**
- 금형 표면 오염
- 이형제 부족
- 금형 마모

**해결 방안**
- 금형 청소 주기 단축 (주 2회 → 주 3회)
- 이형제 균일 도포
- 금형 연마 또는 교체

## 4. 치수 불량 (Dimensional Defects)

### 4.1 수축 과다
**원인**
- 냉각 시간 부족
- 보압 부족
- 용탕 온도 과다

**해결 방안**
- 냉각 시간 증가
- 보압 압력: 90 MPa → 95 MPa
- 용탕 온도 감소

### 4.2 플래시 (Flash)
**원인**
- 클램프 압력 부족
- 금형 정밀도 저하
- 사출 압력 과다

**해결 방안**
- 클램프 압력 증가
- 금형 정비
- 사출 압력 감소

## 5. 크랙 (Crack)

### 5.1 원인
- 급냉으로 인한 열응력
- 취출 시 과도한 힘
- 재료 품질 문제

### 5.2 해결 방안
1. **냉각 조건**
   - 냉각 속도 완화
   - 단계적 냉각 적용

2. **취출 조건**
   - 이젝터 압력 감소
   - 이젝터 핀 개수 증가

3. **재료 관리**
   - 재료 성분 분석
   - 불순물 제거

## 6. 미충전 (Short Shot)

### 6.1 원인
- 사출 압력 부족
- 용탕 온도 부족
- 금형 온도 부족
- 배기 불량

### 6.2 해결 방안
1. **사출 조건**
   - 사출 압력: 120 MPa → 130 MPa
   - 사출 속도 증가
   - 사출 시간 증가

2. **온도 조건**
   - 용탕 온도: 655°C → 670°C
   - 금형 온도: 175°C → 185°C

3. **금형 개선**
   - 배기구 확대
   - 게이트 크기 증가

## 7. 센서 기반 불량 예측

### 7.1 온도 이상 패턴
```
Sensor_Temperature1 > 685°C → 기공 발생 위험 ↑
Sensor_Temperature2 < 170°C → 콜드 샷 위험 ↑
Sensor_Temperature1 - Sensor_Temperature2 > 500°C → 열충격 위험 ↑
```

### 7.2 압력 이상 패턴
```
Sensor_Pressure1 < 115 MPa → 미충전 위험 ↑
Sensor_Pressure2 < 85 MPa → 수축 기공 위험 ↑
Sensor_Pressure1 > 140 MPa → 플래시 위험 ↑
```

### 7.3 진동 이상 패턴
```
Sensor_Vibration > 0.25 g → 기계적 이상
Sensor_Vibration 급증 → 부품 파손 가능
```

4. 품질 데이터 분석

### 8.3 장기 대책 (6개월)
