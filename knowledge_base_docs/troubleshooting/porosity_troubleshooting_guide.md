# 포로시티(Porosity) 불량 트러블슈팅 가이드

**문서 메타데이터:**

- Last Updated: 2024-11-15
- Author: 품질관리팀

## 1. 포로시티 불량 개요

포로시티(기공)는 다이캐스팅 제품 내부에 형성되는 공기 또는 가스 포켓으로, 제품의 기계적 강도를 저하시키고 표면 품질에 영향을 미칩니다.

### 1.1 포로시티 유형
- **가스 포로시티**: 용탕 내 용해된 가스가 응고 중 방출되어 형성
- **수축 포로시티**: 응고 수축으로 인한 공동 형성
- **난류 포로시티**: 주입 중 난류로 인한 공기 혼입

## 2. 주요 원인 분석

### 2.1 공정 파라미터 관련
1. **주입 속도 과다** (Process_InjectionSpeed > 3.0 m/s)
   - 난류 발생으로 공기 혼입 증가
   - 권장 범위: 2.0-2.8 m/s

2. **금형 온도 부족** (Process_MoldTemperature < 160°C)
   - 조기 응고로 가스 배출 불충분
   - 권장 범위: 180-220°C

3. **용탕 온도 과다** (Process_MeltTemperature > 700°C)
   - 가스 용해도 증가
   - 권장 범위: 660-680°C

4. **배압 부족** (Process_BackPressure < 40 bar)
   - 가스 배출 불충분
   - 권장 범위: 45-60 bar

### 2.2 금형 설계 관련
- 배기구 위치 부적절
- 오버플로우 용량 부족
- 게이트 크기 부적절

### 2.3 재료 관련
- 용탕 내 수분 함량 과다
- 이형제 과다 사용
- 재생 알루미늄 비율 과다 (>30%)

## 3. 해결 방법

### 3.1 즉시 조치 사항

#### Step 1: 주입 속도 조정
```
현재 설정 확인:
- Process_InjectionSpeed 값 확인
- 2.5 m/s 이하로 감소
- 5회 샘플 생산 후 품질 확인
```

#### Step 2: 금형 온도 상승
```
목표 온도: 190-200°C
- 예열 시간 연장 (30분 → 45분)
- 냉각수 온도 조정
- Sensor_Temperature2 모니터링
```

#### Step 3: 배압 증가
```
현재 배압 + 10 bar 증가
- 최대 60 bar까지 단계적 증가
- Sensor_Pressure2 모니터링
- 각 단계별 샘플 검사
```

### 3.2 중기 개선 사항

1. **배기구 점검 및 청소**
   - 주기: 매 500 shot
   - 막힌 배기구 청소
   - 필요시 배기구 추가 가공

2. **이형제 사용량 최적화**
   - 현재 사용량의 70%로 감소
   - 분무 패턴 균일화
   - 건조 시간 충분히 확보 (15초 이상)

3. **용탕 품질 관리**
   - 탈가스 처리 강화
   - 용탕 온도 ±5°C 이내 유지
   - 재생 알루미늄 비율 25% 이하 유지

### 3.3 장기 개선 사항

1. **금형 개선**
   - 배기구 위치 재설계
   - 오버플로우 용량 증대
   - 게이트 크기 최적화

2. **진공 다이캐스팅 도입 검토**
   - 투자 비용: 약 5억원
   - 포로시티 발생률 80% 감소 예상
   - ROI: 2-3년

## 4. 품질 검증

### 4.1 검사 방법
- X-ray 검사 (전수 검사)
- 초음파 검사 (샘플링)
- 단면 검사 (파괴 검사)

### 4.2 합격 기준
- 포로시티 크기: 직경 2mm 이하
- 포로시티 개수: 10개/cm² 이하
- 표면으로부터 거리: 3mm 이상
